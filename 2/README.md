3️⃣ 高阶阶段（3~12个月）

目标：掌握 JavaScript 核心原理，能写健壮、可维护的代码。

学习内容：
	•	原型链与继承
	•	this 指向规则（普通函数、箭头函数、bind/apply/call）
	•	闭包、作用域、作用域链
	•	JS 内存管理与垃圾回收
	•	深入理解异步：事件循环、微任务/宏任务
	•	高阶函数、函数式编程基础
	•	模块打包工具（Webpack / Vite）基础
	•	前端框架入门（Vue/React/Angular）
	•	TypeScript 入门（可选，但强烈建议）

难度：高
建议：尝试写组件化代码、小型前端项目，理解框架背后的 JS 原理。
 按照天划分 学习
