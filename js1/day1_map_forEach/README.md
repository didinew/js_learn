1️⃣ map vs forEach（核心）

你要明确这 3 点：

点
map
forEach
是否有返回值
✅ 有新数组
❌ 没有
是否适合数据转换
✅ 非常适合
❌ 不适合
是否改变原数组
❌ 不改变
❌ 不改变（但可手动改）

一句话记忆：

👉 map = 数据转换工具
👉 forEach = 执行副作用
